Name: basis_k
Parameters: N -> An integer. The total number of samples and basis functions.
            k -> An integer. The particular basis function being used.
            n -> A NumPy array. Size corresponds to number of samples taken. Values range from 0
                 to N-1.
Purpose: Calculates the DCT basis function for N samples. The value of k is used to determine
         what basis function is used. In particular, a completely different basis function is
         used when k=0. Returns a NumPy array of results for each sample with a size equivalent
         to n.
